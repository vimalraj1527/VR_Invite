# VR_Invite

A premium, interactive, and highly cultural Tamil Engagement Invitation.
Features:
- 3D Shutter Doors with glowing Vinayagar seal
- Traditional Tamil Thamboolam Trays (Seer Thattu) with hovering animation
- Interactive 3D Engagement Rings animation
- Golden Sparkle Mouse Trail
- Responsive "Save the Date" auto-pull envelope
- Live Event Countdown
- Native "Share Invite" & Google Maps integration
